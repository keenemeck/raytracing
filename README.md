### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/keenemeck/raytracing.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Enter your API in `config.js`
   ```js
   const API_KEY = 'ENTER YOUR API';
   ```
